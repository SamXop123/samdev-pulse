Quick Start Guide

Get up and running in 30 seconds

1

Basic Usage

Simply add this to your GitHub README.md:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME)
2

With Theme

Customize with your favorite color scheme:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME&theme=tokyonight)
3

With LeetCode

Show off your problem-solving prowess:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME&leetcode=YOUR_LEETCODE_USERNAME)
4

Full Customization

Combine all options for maximum impact:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME&theme=dracula&leetcode=YOUR_LEETCODE_USERNAME&align=center)

Available Parameters

Parameter Type Default Description
username string SamXop123 Your GitHub username
theme string dark Theme: dark, light, dracula, nord, tokyonight, monokai, gruvbox
leetcode string - LeetCode username (or "false" to disable)
align string left Header alignment: left, center, right
